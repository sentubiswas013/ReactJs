# 10 Most Asked Azure Interview Questions

## 1. What is Microsoft Azure?

Microsoft Azure is a cloud computing platform that provides:

* Virtual machines
* Storage
* Networking
* Databases
* AI services
* DevOps tools

It supports:

* IaaS
* PaaS
* SaaS

---

## 2. What is the difference between IaaS, PaaS, and SaaS?

| Type | Meaning                     | Example           |
| ---- | --------------------------- | ----------------- |
| IaaS | Infrastructure as a Service | Azure VM          |
| PaaS | Platform as a Service       | Azure App Service |
| SaaS | Software as a Service       | Microsoft 365     |

---

## 3. What is Azure Virtual Machine (VM)?

Azure VM is a scalable compute service used to run:

* Windows servers
* Linux servers
* Applications

Common uses:

* Hosting apps
* Testing
* Development

---

## 4. What is Azure App Service?

Azure App Service is a PaaS service used to deploy:

* Web applications
* REST APIs
* Mobile backends

Benefits:

* Auto scaling
* CI/CD integration
* Managed infrastructure

---

## 5. What is Azure Storage?

Azure Storage is used to store:

* Files
* Blobs
* Queues
* Tables

Types:

* Blob Storage
* File Storage
* Queue Storage
* Table Storage

---

## 6. What is Azure Active Directory (Azure AD)?

Now called Microsoft Entra ID.

Used for:

* Authentication
* Authorization
* Single Sign-On (SSO)
* User management

---

## 7. What is Azure VNet?

Azure Virtual Network allows secure communication between:

* Azure resources
* VMs
* Databases
* On-premise networks

Features:

* Subnets
* NSG
* Peering

---

## 8. What is Azure Load Balancer?

Azure Load Balancer distributes traffic across multiple servers.

Benefits:

* High availability
* Fault tolerance
* Scalability

---

## 9. What is Azure DevOps?

Azure DevOps provides:

* CI/CD pipelines
* Repositories
* Boards
* Test management

Used for automation and deployments.

---

## 10. What is ARM Template?

ARM (Azure Resource Manager) template is Infrastructure as Code (IaC) for Azure.

Written in JSON.

Used to automate:

* VM creation
* Networking
* Storage
* Entire infrastructure

---
